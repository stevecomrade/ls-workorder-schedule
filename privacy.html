Privacy Policy

Last updated: April 2026

Workorder Schedule for Lightspeed is a browser extension that connects to the Lightspeed Retail R-Series API to display your shop's workorder schedule. This privacy policy explains what data is accessed and how it is handled.

What data is accessed

The extension accesses workorder data from your Lightspeed Retail account via the official Lightspeed API. This includes workorder details, customer names, employee names, and workorder line items. This data is used solely to display your workorder schedule within the extension.

How data is stored

All data is processed locally in your browser. No workorder data, customer data, or any other Lightspeed data is transmitted to any third-party server or stored outside your browser. API credentials (your Account ID, Client ID, Client Secret, and OAuth tokens) are stored locally in your browser's extension storage and are never sent to anyone other than Lightspeed's API servers.

OAuth credentials

The extension uses OAuth 2.0 to authenticate with Lightspeed on your behalf. Your credentials are stored locally in browser storage and used only to make API requests to Lightspeed. The OAuth callback page is hosted on GitHub Pages solely to receive the authorization code and display it to you — no data is logged or stored by that page.

Third parties

The extension communicates only with Lightspeed's API servers (api.lightspeedapp.com and cloud.lightspeedapp.com). No data is shared with any other third party.
